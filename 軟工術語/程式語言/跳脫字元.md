## 跳脫字元

當你想要 跳脫某個英文字母的字面意義、表現/編碼符號，或使用被系統保留的特殊符號 時，可以使用跳脫字元。

* C#、C、Python... 跳脫字元為 `\`。例如：
  * `\n`：換行
  * `\x23` ：`x23` 是 `#` 的十六進位表示
  * `\\` ：反斜線

* URI 跳脫字元為 `%`
  * `20` 是空白格的 16 進位 表示
```
var shortName = @"./軟工術語/一般/URI VS URL.md";
var path = shortName.Replace(" ","%20"); 

// path = @"./軟工術語/一般/URI%20VS%20URL.md
```

* `@` 可以跳脫整個字串序列，例如：
  * path = @"C:\OnlyForTest\Folder1";
