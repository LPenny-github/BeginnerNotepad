# Keyboard shortcuts


## 視窗調整

* `Windows` + `↑` / `↓` : 視窗展開（Maximize） / 視窗還原成原本大小（Restore Down）

* `alt` + `space` : 放大、縮小、還原......等等視窗快捷目錄

  * 把該視窗縮到最小（Miniimize）
    * `alt` + `space`  `N`

  * 把全部視窗縮到最小，露出桌面

    （**雖然以下兩個指令據說效果不同，但我試了幾回目前看不出有何差異**）

    * `Windows` + `D` 
    * `Windows` + `M`
