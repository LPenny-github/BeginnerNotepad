# 什麼是 JSON？

* JSON，JavaScript Object Notation。
  * 一種 資料/物件 的格式
  * 許多程式語言都有支援
    * > 即便它源自JavaScript，但目前很多程式語言都支援 JSON 格式資料的生成和解析。

* 注意
  * JSON 傳遞數值，不能包含 NaN
    * > 數值：十進位數，不能有前導0，可以為負數，可以有小數部分。還可以用e或者E表示指數部分。不能包含非數，如NaN。不區分整數與浮點數。JavaScript用雙精度浮點數表示所有數值。
