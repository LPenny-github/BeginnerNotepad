#快速鍵
 

* `ctrl` + `k` , `ctrl` + `S` : keyboard shortcus

* `ctrl` + `shift` + `N` : 開新的 VS Code

* `ctrl` + `B` : 側欄開啟或關閉

* `ctrl` + `↑` / `↓` : 畫面上 / 下滾動

* `ctrl` + `shift` + `K` : 刪除該行

* `shift` + `tab` : 往前縮排
