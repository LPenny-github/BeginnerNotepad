# 更改 Repo 名稱

## 在 GitHub 更改 Repo 名稱

1. 進入目標 Repo
2. 點 `Settings`
3. 在 Repository name 鍵入 Repo 的新名字，按 `Rename`

## 在 VS Code 更新 URL（若需要）

1. （在該 Repo 中 TERMINAL）鍵入指令： git remote set-url origin **new_url**
    * 例如： git remote set-url origin git@github.com:LPenny-github/Notepad.git
