# 避免網址被自動翻譯成16進位

## 範例

* 肉眼所見的網址： https://zh.wikipedia.org/wiki/转义字符
* 點擊 + 複製 + 貼上 後的網址：https://zh.wikipedia.org/wiki/%E8%BD%AC%E4%B9%89%E5%AD%97%E7%AC%A6

如果要避免被翻譯，最簡單的方法就是 **不要複製全部網址**，例如：你可以技術性少複製最前面那個 `h`（`h`自己用鍵盤打）。

## 注意

1. 不是每個網站、每個網址 都會自動被翻譯成16進位。

像是上面網址的英文版：https://en.wikipedia.org/wiki/Escape_character

就是 `肉眼所見` 和 `點擊 + 複製 + 貼上` 後的網址 相同。

2. （依據各網站的情況有別）網址所呈現的中文，未必是 `繁體中文`。

以上面的網址為例，繁體中文是 `跳脫字元`，簡體中文是 `转义字符`。

雖然，不想讓網址被翻譯成 16 進位就是想避免「猜」網址的狀況發生，但有時還是不免需要動用通靈能力。
